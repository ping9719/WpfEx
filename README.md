### 快速开始 [How To Use]
一、在App.xaml中添加以下代码：
```CSharp
    <Application.Resources>
        <ResourceDictionary>
            <ResourceDictionary.MergedDictionaries>
                <ResourceDictionary Source="pack://application:,,,/Ping9719.WpfEx;component/Theme.xaml"/>
            </ResourceDictionary.MergedDictionaries>
        </ResourceDictionary>
    </Application.Resources>
```
一、添加命名空间：
```CSharp
xmlns:pi="https://github.com/ping9719/wpfex"
```
